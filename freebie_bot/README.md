# 🎁 Telegram Freebie Bot (Авто-Халява)

Этот бот автоматически собирает 100% скидки и бесплатные раздачи игр со всего интернета (Reddit, Epic Games Store) и публикует их в ваш Telegram-канал. 

**Главная фишка:** Проект работает абсолютно **БЕСПЛАТНО** на серверах GitHub Actions. Вам не нужно платить за хостинг!

---

## 🚀 Инструкция по запуску (Занимает 5 минут)

### Шаг 1: Создание Telegram Бота
1. Зайдите в Telegram и найдите бота [@BotFather](https://t.me/BotFather).
2. Напишите ему команду `/newbot`.
3. Придумайте имя (например, `Халява Бот`) и юзернейм (обязательно заканчивается на `bot`, например `freebie_super_bot`).
4. **Скопируйте Токен бота** (длинная строка вида `123456789:ABCdefGHI...`). Никому его не показывайте!

### Шаг 2: Создание Telegram Канала
1. Создайте публичный канал в Telegram.
2. Придумайте ему ссылку (например, `@super_halyava_2026`).
3. Зайдите в настройки канала -> **Администраторы** -> Добавить администратора.
4. Найдите вашего бота по юзернейму (из Шага 1) и добавьте его в админы (разрешите ему отправлять сообщения).

### Шаг 3: Загрузка на GitHub
1. Зарегистрируйтесь на [GitHub.com](https://github.com/), если у вас еще нет аккаунта.
2. Создайте новый репозиторий (New Repository), назовите его, например, `freebie-bot`. **Обязательно сделайте его Private (Приватным)**.
3. Загрузите в этот репозиторий все файлы из этой папки (особенно `main.py`, `requirements.txt`, `posted_ids.json` и папку `.github`).

### Шаг 4: Настройка секретов (Secrets)
Гитхаб должен знать токен вашего бота, но хранить его в коде небезопасно.
1. Зайдите в вашем репозитории на GitHub во вкладку **Settings** (Настройки).
2. Слева в меню найдите **Secrets and variables** -> **Actions**.
3. Нажмите зеленую кнопку **New repository secret**.
4. Добавьте первый секрет:
   - Name: `TELEGRAM_BOT_TOKEN`
   - Secret: вставьте ваш токен из Шага 1.
5. Нажмите "Add secret" и создайте второй:
   - Name: `TELEGRAM_CHAT_ID`
   - Secret: вставьте @имя_вашего_канала (с собачкой, например `@super_halyava_2026`).

### Шаг 5: Вы великолепны ✨
Всё! Бот настроен. 
Зайдите во вкладку **Actions** в GitHub, выберите слева "Freebie Bot Scheduler", нажмите "Run workflow" -> зеленая кнопка "Run workflow". 
В течение минуты скрипт запустится и пришлет первые посты в ваш канал. После этого он будет запускаться **автоматически каждый час**, искать новые игры и сразу постить их.

## 🛠 Технические особенности
* **Парсеры:** Использует официальный GraphQL API Epic Games и JSON API Reddit.
* **Анти-Спам:** Бот хранит список отправленных игр в файле `posted_ids.json`. GitHub Actions после каждого запуска делает автоматический коммит обновленного файла обратно в репозиторий.
* **Фильтрация:** Встроенные фильтры отсеивают мусор (дискуссии, неполные скидки), оставляя только 100% Game Deals.
